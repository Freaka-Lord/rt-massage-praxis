# RT Massage Praxis — Modern Website (2026)

Eine moderne, responsive Single-Page-Website für die **RT Massage Praxis** in Schaffhausen.

## Features

- Modernes 2026-Design mit Golden-Cream-Farbpalette
- Vollständig responsive (Mobile-First)
- Interaktive Bildergalerie mit Lightbox
- OpenStreetMap Integration (Leaflet)
- Scroll-Animationen (AOS)
- Google Bewertungen Integration (4,9 ★)
- Floating WhatsApp/Telefon CTA
- PWA-ready (manifest.json)
- SEO-optimiert

## Technologie-Stack

- Vanilla HTML5 / CSS3 / JavaScript
- Leaflet.js (Karte, kein Google API-Key nötig)
- AOS (Scroll-Animationen)
- Font Awesome Icons
- Cormorant Garamond & Montserrat Fonts
- Cloudflare-ready

## Deployment auf GitHub Pages

```bash
# Repository erstellen
git init
git add .
git commit -m "Initial commit"

# Auf GitHub pushen
gh repo create rt-massage-praxis --public --source=. --remote=origin --push

# GitHub Pages aktivieren:
# Settings → Pages → Branch: main → / (root)
```

Oder einfach per FTP aufs Webhosting kopieren.
